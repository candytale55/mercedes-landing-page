## HTML Structure

HEADER and NavBars:

- nav.main-nav (for the primary navigation with logo and menu items)
- ul.icon-nav-list (for the secondary navigation with user/cart/menu icons)
- brand-logo (more descriptive for the Mercedes logo)
- ul.main-nav-list (clearer indication that it's the main navigation list)
- a.nav-link (anchor tags inside ul>li)
